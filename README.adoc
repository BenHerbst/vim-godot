== Folding

To enable folding, `setlocal foldmethod=expr` in your `.vimrc`.

[source,vim]
------------------------------------------------------------------------------
augroup godot | au!
    au FileType gdscript setlocal foldmethod=expr
augroup END
------------------------------------------------------------------------------
